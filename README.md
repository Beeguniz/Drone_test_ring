# Multi-Axis UAV Test Bench

## Overview

This project is a UAV Test Bench designed for evaluating quadcopter flight controllers in a controlled environment. The mechanical structure was developed based on existing UAV test bench concepts and reference designs, then modified and reconstructed in SolidWorks to suit larger UAV platforms and specific project requirements.

The platform enables safe testing of attitude control algorithms, PID tuning, and sensor validation without conducting actual flight tests.

## My Contributions

* Recreated the mechanical design in SolidWorks.
* Modified structural dimensions to support larger UAV platforms.
* Redesigned mounting components and assembly structure.
* Generated complete CAD models and assembly files.
* Prepared the design for future manufacturing and testing.

## Features

* Multi-axis rotational mechanism
* Modular assembly design
* Support for F450-class UAVs and larger
* Bearing-supported rotational joints
* Suitable for flight controller development and validation
